# DTW

  -Dynamic time wrapping 알고리즘을 이용한 4개의 음악 허밍 데이터를 이용해 각 음악의 유사도를 판별하는 프로그램입니다.
  
  -실행 자체에 오랜 시간이 걸리기 때문에 실행해보는 것을 권장하지 않습니다.
